# IseuSchedule V2

Android-приложение для просмотра расписания занятий, работы с личным кабинетом и получения уведомлений о парах.

Проект ориентирован на студентов и преподавателей, которым нужен быстрый доступ к актуальному расписанию, информации об успеваемости и базовым данным учебного кабинета прямо с Android-устройства.

> ⚠️ Проект не является официальным приложением учебного заведения. Все названия, домены и данные используются только для взаимодействия с открытыми/доступными пользователю сервисами.

---

## Возможности

- Просмотр расписания занятий.
- Режим студента и режим преподавателя.
- Авторизация и работа с пользовательской сессией.
- Просмотр информации об успеваемости.
- Локальное кэширование данных.
- Настройки приложения.
- Push/local-уведомления о занятиях.
- Восстановление уведомлений после перезагрузки устройства.
- Обработка HTML-страниц расписания.
- OCR-распознавание капчи через Google ML Kit.
- Современный UI на Jetpack Compose.

## Архитектура проекта

Проект разделён на несколько основных слоёв:

```text
app/
└── src/main/java/com/example/scheduleiseu/
    ├── app/              # Корневые компоненты приложения
    ├── core/             # Общие утилиты, модели, расширения
    ├── data/             # Источники данных, репозитории, парсеры, БД
    │   ├── local/        # Room, cache, preferences
    │   ├── remote/       # Network/data sources/parsers
    │   ├── repository/   # Реализации репозиториев
    │   ├── mapper/       # Маппинг моделей
    │   ├── ocr/          # OCR-обработка капчи
    │   └── session/      # Работа с пользовательской сессией
    ├── domain/           # Доменная логика и контракты
    ├── feature/          # UI-фичи приложения
    │   ├── auth/         # Авторизация
    │   ├── home/         # Главный экран и расписание
    │   ├── performance/  # Успеваемость
    │   ├── settings/     # Настройки
    │   └── navigation/   # Навигация
    ├── notification/     # Уведомления и alarm receiver
    └── MainActivity.kt
